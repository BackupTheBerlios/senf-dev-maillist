On 02/01/2010 06:58 PM, Stefan Bund wrote:
[...]
> There is an additional overload of fty::Command. The correct
> incantation now is
>
>     namespace fty = senf::console::factory;
>     scopeddir.add("foo", fty::Command(this,&memberfunction));

after some longer discussion with no final conclusion I've decided to 
change to API slightly again. To be consistent with senf::membind the 
parameters of factory::Command will be changed.

Old:
dir.add("foo", fty::Command(this, &memberfunction));

New:
dir.add("foo", fty::Command(&memberfunction, this));
