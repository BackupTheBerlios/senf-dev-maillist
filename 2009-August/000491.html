Hi everyone,

while refactoring the build system, I have renamed the 'senfscons'
folder to 'site_scons', which is the SCons default name for such a
project specific library folder. This folder is automatically added to
the python search path when calling scons.

Since external projects which use SENF load the 'senfutil' module from
'senfscons' (now 'site_scons'), those external projects need to change
their python path accordingly.

SUMMARY --------------------------------------------------------------

    Rename  

        'senfscons' -> 'site_scons' 

    in all external Projects using SENF

END SUMMARY ----------------------------------------------------------
