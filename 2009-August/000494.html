Hi everyone,

If you are using senfutil to build a project depending in SENF, you
will have to update the SConstruct file a little bit after the
following commit. Just look over the new documentation in

    <Senf-root>/doc/html/senf_senfutil.html

(after re-building the documentation with 'scons doc'). Most projects
will probably replace most SCons environment variable definitions with
a single call to

    senfutil.DefaultOptions(env)
