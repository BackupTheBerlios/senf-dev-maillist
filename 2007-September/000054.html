Hi all,

I found out, that scons changed an important detail between 0.96.1
(the Version which seems to be part of ubuntu and other distributions)
and some later Version (before 0.96.93, the Version I am running). In
essence, several constructs are a little bit differnet, and one of the
visible things is, that actions bound to aliases don't work in 0.96.1
as they do in 0.96.93. That's the reason, why 'scons deb' does
nothing in ubuntu.

I have fixed the problem by backporting the SConscript file using some
ugly hacks ...
