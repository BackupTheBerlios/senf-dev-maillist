Hi everyone,

I have updated the nightly build system somewhat:
habe jetzt das senf nightly build system etwas angepasst:

 - Nightly incremental builds are performed in the nights Mo/Tu to
   Fr/Sa.

 - A complete build occurs in the night Sa/So.

 - If that complete build is successful, snapshot packets for all
   registered distributions are automatically built. Currently, these
   are  Ubuntu dapper, gutsy, and hardy as well as debian etch.

I'll soon update the debian build to execute the unit tests. Thus we
get a periodic and automatic test of SENF of several linux platforms
and will hopefully find missing dependencies or version problems
faster.
