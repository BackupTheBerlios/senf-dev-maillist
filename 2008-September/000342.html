Hi everyone,

I have now comited the finalize() API changes. Instead of a single
finalize() member, packets now have the members:

 - finalizeAll(): The same as the old finalize()

 - finalizeThis(): Finalize only the current header

 - finalizeTo<Other>(): Finalize until first occurence of packet type
   Other

 - finalizeTo(other): Finalize until packet other
