We (Mathias and me) have been doing some compatibility checks with
SENF. We have tested the following combinations (Mathias, please
correct me if I'm wrong :-) ). All these combinations work now:

 - ia32 gcc 4.1 with boost version 1.34.1, 1.35 and 1.36 (on debian stable)
 - ia32 gcc 4.3 with boost version 1.35 and 1.36 (on ubuntu intrepid)
 - ia64 gcc 4.3 with boost version 1.35 (on gentoo)

I don't think there will be dramatic difficulties with other
combinations.

stefan.
