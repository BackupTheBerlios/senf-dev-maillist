Hi everyone,

as discussed on the last meeting, I have changed the
senf::ppi::connect() API regarding modules with multiple dynamic
connectors (AnnotationRouter, Joins etc).

OLD API:  senf::ppi::connect(source, target, source-args, target-args)
NEW API:  senf::ppi::connect(source, source-args, target, target-args)

('source-args' and 'target-args' are both optional). You need to move all
'source-args' *before* the 'target' argument. Failing to do so will
give you horrible error messages ...

I have NOT commited this chnage just now. I will do so, as soon as I
get the go-ahead.

stefan.
