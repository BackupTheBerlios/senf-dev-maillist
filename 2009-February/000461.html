Hi everyone,

I have just checkid in a hires timer implementation based on the linux
timerfd() syscall. This version should automatically be used when
available:

 * GNU-Libc library version >= 2.8
 * Linux kernel version >= 2.6.25

When using hires timers, you should not see any difference appart from
a reduced signal load. We need to test the timerfd() performance in
comparision to the POSIX timer implementation.

stefan.
