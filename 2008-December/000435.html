As I wrote in the bug comment, I don't like the toString() member, use

  boost::lexical_cast<std::string>(...) 

instead.

However, if everyone agrees, that this is to complicated, we could add
a simple alias

  senf::str(...)

which just calls boost::lexical_cast<std::string>(...). The advantage
is, that this works for *any* object.
